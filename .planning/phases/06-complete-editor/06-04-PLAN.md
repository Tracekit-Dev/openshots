The CropOverlay is a Konva Layer that renders:
1. Four darkened rects forming a mask around the crop box (rgba(0,0,0,0.5) per D-02)
2. The crop box border (1px white, per UI-SPEC)
3. Eight handles (4 corners + 4 edges) as small white Rect nodes (8x8px visual, draggable)
4. Aspect ratio constraint logic

Props:
```typescript
interface CropOverlayProps {
  image: CanvasImage;           // The selected image being cropped
  canvasWidth: number;
  canvasHeight: number;
  onConfirm: (cropRect: { x: number; y: number; width: number; height: number }) => void;
  onCancel: () => void;
}
```

Implementation details:

1. Local state for crop rect initialized to the full image bounds:
   ```typescript
   const [cropRect, setCropRect] = useState({ x: imgLeft, y: imgTop, width: imgW, height: imgH });
   const [aspectRatio, setAspectRatio] = useState<number | null>(null); // null = free
   ```
   Where imgLeft = image.x - image.width/2, imgTop = image.y - image.height/2, etc.

2. Aspect ratio presets per D-04:
   ```typescript
   const ASPECT_RATIOS: { label: string; value: number | null }[] = [
     { label: "Free", value: null },
     { label: "16:9", value: 16 / 9 },
     { label: "4:3", value: 4 / 3 },
     { label: "1:1", value: 1 },
   ];
   ```

3. Handle drag logic: Each handle adjusts the crop rect. When aspect ratio is set, constrain:
   ```typescript
   function constrainToRatio(w: number, h: number, ratio: number | null): { width: number; height: number } {
     if (!ratio) return { width: w, height: h };
     if (w / h > ratio) return { width: h * ratio, height: h };
     return { width: w, height: w / ratio };
   }
   ```

4. Mask rendering: Four Rect nodes around the crop box:
   - Top: y=0, height=cropRect.y
   - Bottom: y=cropRect.y+cropRect.height, height=canvasHeight-(cropRect.y+cropRect.height)
   - Left: y=cropRect.y, height=cropRect.height, width=cropRect.x
   - Right: y=cropRect.y, x=cropRect.x+cropRect.width, height=cropRect.height

5. Crop box border: Rect with no fill, stroke="rgba(255,255,255,0.8)", strokeWidth=1

6. Handles: 8 Rect nodes at corners and edge midpoints, each 8x8px, fill="#ffffff", draggable.
   Corner handles: top-left, top-right, bottom-left, bottom-right
   Edge handles: top-center, bottom-center, left-center, right-center
   Each handle has an onDragMove that adjusts cropRect accordingly.

7. Aspect ratio buttons are rendered as HTML (not Konva) via a React portal or an absolutely positioned div. Since this is inside a Konva Stage, use the `Html` pattern: render the buttons as a fixed-position HTML overlay outside the Stage. This means the CanvasStage parent must render the aspect ratio bar.

Actually, simpler approach: render the aspect ratio bar and confirm/cancel buttons as HTML elements OUTSIDE the Stage in CanvasStage.tsx, only when crop mode is active. The CropOverlay Konva layer handles just the mask + handles.

So CropOverlay.tsx renders the Konva layer:
```tsx
export default function CropOverlay({ image, canvasWidth, canvasHeight, cropRect, setCropRect, aspectRatio }: CropOverlayProps) {
  // ... mask rects, border rect, handle rects
  return <Layer>...</Layer>;
}
```

And the aspect ratio bar + confirm/cancel are rendered in CanvasStage.tsx as HTML.

The onConfirm callback (called from CanvasStage on Enter or button click):
- Creates an offscreen canvas
- Loads the image.src
- Draws the cropped region: source rect = crop box mapped to image pixel space
- Converts to data URL
- Calls updateImage(id, { src: newDataUrl, width: cropW, height: cropH, cornerRadius: 0 })

Full CropOverlay component should include:

5. Add crop confirm handler:
   ```typescript
   const handleCropConfirm = useCallback(() => {
     if (!cropRect || !selectedImage) return;
     const img = new window.Image();
     img.crossOrigin = "anonymous";
     img.src = selectedImage.src;
     img.onload = () => {
       // Map crop rect from canvas space to image pixel space
       const imgLeft = selectedImage.x - selectedImage.width / 2;
       const imgTop = selectedImage.y - selectedImage.height / 2;
       const scaleX = img.naturalWidth / selectedImage.width;
       const scaleY = img.naturalHeight / selectedImage.height;
       const sx = Math.round((cropRect.x - imgLeft) * scaleX);
       const sy = Math.round((cropRect.y - imgTop) * scaleY);
       const sw = Math.round(cropRect.width * scaleX);
       const sh = Math.round(cropRect.height * scaleY);

       const canvas = document.createElement("canvas");
       canvas.width = sw;
       canvas.height = sh;
       const ctx = canvas.getContext("2d")!;
       ctx.drawImage(img, sx, sy, sw, sh, 0, 0, sw, sh);
       const newSrc = canvas.toDataURL("image/png");

       updateImage(selectedImage.id, {
         src: newSrc,
         width: cropRect.width,
         height: cropRect.height,
         x: cropRect.x + cropRect.width / 2,
         y: cropRect.y + cropRect.height / 2,
       });
       setActiveTool("select");
       setCropRect(null);
     };
   }, [cropRect, selectedImage, updateImage, setActiveTool]);
   ```
